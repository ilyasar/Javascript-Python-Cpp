Что такое Javascript?

Javascript - это язык программирования, который используется для игр на сайтах, анимаций, рекламы, кнопок. Один из самых востребованых языков программирования. Используется на Web - страницах или сайтах.

Как с ним работать?

Можно даже и в поисковике набрать about:blank, нажать f11 и выбрать console вот и все! Но мы будем использовать более сложный вариант. Открываем блокнот и сохраняем файл с расширением <.html>. Готово!

А теперь напишем простую программу с использованием кнопки выше. 

<head>
    <title>Hello World!</title>
</head>
<body>
    <script>
        var message = function(){
            alert("Hello World!")
        }
    </script>
    <button onclick="message();">Нажми меня!</button>
</body>

Как запустить?

Сохраните файл, затем в браузере, нажмите сочетание клавиш ctrl + o и выберите этот файл. Кликните по кнопке и... готово!
